CIELUV 1976 chromaticity diagram web-based app

Author: E H H Hasabeldaim

Author affiliation: Department of Physics, University of the Free State, Bloemfontein, 9300, South Africa

Published: 2021

Abstract:

CIELUV 1976 uniform chromaticity diagram app is been develpoed and rigorously tested. The software is an online and user-friendly application that works in most modren browsers for all systems (windows, mac, and linux). The user will be able to upload a text file containing numerical data of a photoluminescence, electroluminescence, cathodoluminescence, or light emitting diode (LED) emission spectrum, and simply calculate the CIELUV parameters (u' and v'), plot the results in the color space gamut and save the final results as an png file format (image). This online software application is useful for scientists, professionals, and engineers in the fields of luminescence, LEDs, displays, color science, and graphics.

Features:

This app allow the user to

  1. upload photoluminescence, cathodoluminescence, electroluminescence or light emitting diode (LED) emission spectrum data in text format (comma/ tab delimited ).
  2. Calculate the u' v' coordinates.
  3. Add multiple CIELUV points on the CIELUV diagram.
  4. Configure the diagram layout including the axis width, color, font, font size.
  5. Save the final diagram as png file format.

References:

  1. Schanda, Janos. "CIE colorimetry." Colorimetry: Understanding the CIE system (2007): 25-78.
  2. Parkyz, Du-Sik, Jong-Seung Parky, and Joon Hee Hany. "Image indexing using color histogram in the CIELUV Color Space." (1999).
