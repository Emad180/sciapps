CIELUV 1976 chromaticity diagram web-based app

By E H H Hasabeldaim

Department of Physics, University of the Free State, Bloemfontein, 9300, South Africa

Published 2021

Abstract

CIELUV 1976 uniform chromaticity diagram app has been develpoed and rigorously tested. The software is an online and user-friendly application that works in most modren browsers for all systems (windows, mac, and linux). The user will be able to upload a text file containing numerical data of a photoluminescence, electroluminescence, cathodoluminescence, or light emitting diode (LED) emission spectrum, and simply calculate the CIELUV parameters (u' and v'), plot the results in the color space gamut and save the final results as an png file format (image). This online software application is useful for scientists, professionals, and engineers in the fields of luminescence, LEDs, displays, color science, and graphics.

Features

This app allow the user to

  1. upload photoluminescence, cathodoluminescence, electroluminescence or light emitting diode (LED) emission spectrum data in text format (comma/ tab delimited ).
  2. Calculate the u' v' coordinates.
  3. Add multiple CIELUV points on the CIELUV diagram.
  4. Configure the diagram layout including the axis width, color, font, font size.
  5. Save the final diagram as png file format.

Necessary information for accurate calculations:

  1. The color matching function (data) used is 1931 2 deg.
  2. Data wavelength range from 360 to 830 nm.
  3. User data wavelength interval can be 1 nm, 2 nm, or 5 nm for the best estimate. However, any wavelenth interval will give reasonable estimate
  4. You may use text/csv file depending on you browser.
  5. Download test file from Here.txt or Here.csv
